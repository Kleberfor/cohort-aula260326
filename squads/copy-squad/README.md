# copy-squad

Squad didático de copywriting para páginas de alta conversão.

## Agente

- **Cora** (`@copy`) — Copywriter especialista em conversão

## Uso

```
@copy
*write-page
```

## Comandos

| Comando | Descrição |
|---------|-----------|
| `*write-page` | Copy completo de uma página |
| `*write-headline` | Variações de headline para A/B test |
| `*write-cta` | CTAs persuasivos |

## Estrutura

```
copy-squad/
├── squad.yaml          # Manifest
├── README.md           # Este arquivo
├── agents/
│   └── copy.md         # Agente Cora
└── tasks/
    └── copy-write-page.md  # Task principal
```
